版本号：CW Pokemon Mini(Pro) V5.01


用户升级固件       （必须刷入）
系统资源文件       （无须刷入）
开机画面文件       （无需刷入）
四合一字库文件     （无需刷入）



A000014.bin        用户升级固件
F000000.bin        开机画面文件（用户可自定义，非升级必须文件，看ReadMe说明操作）
F0000C0.bin        系统资源文件（非升级必须文件，看ReadMe说明操作）
F003800.bin        四合一中文字库文件（非升级必须文件，看ReadMe说明操作）


固件升级教程
1. 按住F1和F4键的同时开机，LED亮后松开按键；
2. 确认屏幕显示升级提示信息，如果没有升级信息，请重复步骤1操作；
3. 使用数据线连接设备到电脑，电脑提示发现一个名为HAM MODULE的移动存储设备;
4. 根据升级包的说明，将相应的文件复制移动存储设备中，一次只能复制一个文件，复制下一个文件前，需要长按F4使得移动存储设备复位一次，然后再继续复制其它的升级文件(文件名不得擅自更改)；
5. 所需升级文件复制完毕后，长按电源键或者使用细针按压复位孔内的复位键关闭设备，等待3-5秒后重启设备，完成升级工作。

Upgrade instructions


Version Number:  CW Pokemon Mini(Pro) V5.00


User Upgrade Firmware                (Must programming)
System resource files                (no need to programming)
Files of boot screen                 (no need to programming)
4 in 1 font file                     (no need to programming)



A000014.bin                          user upgrades firmware
F000000.bin                          boot screen file (user can be customized, not a necessary file to upgrade, see README operation)
F0000c0.bin                          system resource file (not required for upgrade, see README operation)
F003800.bin                          4-in-1 Chinese character library file (not a required file for upgrade, see README operation)


Firmware Upgrade Tutorial
1. Press F1 and F4 keys while starting the machine, and release the button when the LED lights up;
2. Make sure the screen displays the upgrade message. If there is no upgrade message, repeat Step 1;
3. Connect the device to the computer with the data cable, and the computer prompts to find a mobile storage device named Ham Module;
4. Copy the corresponding files to the mobile storage device according to the instructions of the upgrade package. Only one file can be copied at a time. Before copying the next file, long press F4 to reset the mobile storage device for one time, and then continue to copy other upgraded files (file name shall not be changed without authorization);
5. After copying the required upgrade files, long press the power button or press the reset button in the reset hole with a fine needle to turn off the device, wait for 3-5 seconds and restart the device to complete the upgrade.
