/***  Release Notes for BETA  ***/
2024/12/28     V5.03.11
修正随机生成电码有重复几率的bug。
Fixed a bug where randomly generated codes had a chance to be repeated.


2024/08/07     V5.03.10
修复了跟报练习中电键输入错误3次后，下一个新字符会重复显示上一个错误字符的bug。
Fixed a bug in the step by step mode where the next new character would repeat the previous wrong character after 3 key input errors.

2024/07/08     V5.03.09
在跟报练习模式下，增加输入错误超过3次自动跳转下一个字符功能，避免用户在一个字符上反复练习，无法中断退出。
In the step by step mode, add the function of automatically jumping to the next character when the input error is more than 3 times, so as to prevent the user from repeatedly practicing on a character and unable to interrupt the exit.

2023/07/14     V5.03.08
修复Iambic A模式下，电键点划交换后电键控制异常的bug；CW发报练习中，在混合模式下增加"/ ? ="三个通联常用字符，同时提高了其出现的几率。
Fixed A bug in Iambic A mode where the key control was abnormal after the key dot swap.
In the CW Posting exercise, add "/?="in mixed mode. Three commonly used characters are linked, while increasing the probability of their occurrence.

2022/11/13     V5.03.07
修复了随机生成电码报底重复的bug；听抄练习开始前增加3个字母V作为练习前导符。
Fixed the bug of randomly generating code bottom repeats. Add three letters "V" as a lead before the listening and copying exercise begins.


2022/10/23     V5.03.06
修复了随机生成电码报底重复的bug；修改英文数字0显示时添加斜杠，以便跟字母O区分开来；增加参数设置时，当前设置内容反色显示功能。

Fixed the bug of randomly generating code bottom repeats. Change the display of English number 0 to add a slash to distinguish it from the letter O; When adding parameter Settings, the current setting of the contents of the reverse color display function.

2022/02/14     V5.03.05
修复了通联日志使用电键录入时，最后一个字符被空格覆盖的bug。

Fixed a bug where the last character was overwritten by a space when the log was entered using a key.


2022/01/20     V5.03.04
修复了听抄训练结果符合时，第一页以后数据错位8行的bug；
修复了跟报训练字符0不显示的bug。

Fixed a bug where 8 lines of data are misplaced after the first page when the results of listening and copying training match.  
Fixed a bug where training character 0 would not be displayed.
